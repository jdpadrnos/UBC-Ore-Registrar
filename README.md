* Big Reactors (INCOMPLETE)
* Biomes O Plenty
* BluePower
* Factorization
* Farlanders
* Flaxbeard's Steam Power
* Forestry
* HarvestCraft
* IC2
* Magical Crops
* Mekanism
* Project:RED
* Railcraft (Sulfur)
* SteamCraft
* Thermal Foundation
* Thaumcraft
